The [4P-CAN Project](https://4p-can.eu/) is an EU HORIZON funded project conducting personalized cancer primary prevention research through citizen participation. Among others respondents' personal networks have been collected. The configuration file and R script linked below specify and estimate RHEM explaining alter-nominations. 
The configuration file [`alter_nomination_config.txt`](https://github.com/juergenlerner/eventnet/blob/master/data/4P-CAN_alter_nominations/alter_nomination_config.txt) can be loaded in eventnet and computes hyperedge covariates based on the input file `events.csv` (ADD LINK TO REPOSITORY). The resulting output file can be used to estimate RHEM as sketched in the R code file [`alter_nomination_model_estimation.R`](https://github.com/juergenlerner/eventnet/blob/master/data/4P-CAN_alter_nominations/alter_nomination_model_estimation.R)
