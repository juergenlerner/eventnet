The EU HORIZON funded [4P-CAN Project](https://4p-can.eu/) conducts personalized cancer primary prevention research through citizen participation. It collects respondents' personal networks, along with several demographic and behavioral attributes of egos and alters over several waves of data collection. The configuration file and R script linked below specify and estimate RHEM explaining the selection of the set of alters by an ego, based on exogenous attributes on actors, dyads, or groups, previous nominations by the same or different respondents, joint participation in social events, and explicit links between actors.

The configuration file [`alter_nomination_config.txt`](https://github.com/juergenlerner/eventnet/blob/master/data/4P-CAN_alter_nominations/alter_nomination_config.txt) can be executed with eventnet, computing hyperedge covariates based on the input file `events.csv` available at [https://zenodo.org/records/21409040](https://zenodo.org/records/21409040). The resulting output file can be used to estimate RHEM as sketched in the R code file [`alter_nomination_model_estimation.R`](https://github.com/juergenlerner/eventnet/blob/master/data/4P-CAN_alter_nominations/alter_nomination_model_estimation.R)
